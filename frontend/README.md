# HiveWatch Lite React Front End

A React and TypeScript front end for managing hives and temperature readings in the HiveWatch Lite system.

## Features

- View all hives
- Add, edit, and delete hives
- View all temperature readings
- Add, edit, and delete temperature readings
- Reassign a temperature reading to a different hive

## Running locally

1. Start the Spring Boot back end on `http://localhost:8080`
2. Install dependencies:

```bash
npm install
```

3. Start the React development server:

```bash
npm run dev
```

4. Open the local URL shown in the terminal. It is usually:

```text
http://localhost:5173
```
